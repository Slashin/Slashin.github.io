This is my current website in which I decided to go with a more calm and light-colored look. I also reduced my color scheme and provided my site with a more properly themed effect, based on human psychology and visual interpretation. As for the programming aspect, I was able to drastically improve my core fundamentals and also explore newer areas of development. For example, I had a lot of fun learning how to create the 3D carousel and card-flip effects with the use of 3D CSS. I was also able to learn newer technologies such as JQuery and went deeper into the properties of Bootstrap!

Tools Used: HTML, CSS, JavaScript, Bootstrap, JQuery
