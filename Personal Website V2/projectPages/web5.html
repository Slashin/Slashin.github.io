This project was the first time that I used JavaScript for purposes other than simply adding interactivity to websites. In this application, there is very little usage of HTML and CSS, but a lot of emphasis on JS. By creating classes, objects and multiple methods, I was able to create a proper UI for a simple daily planner. I also learned a lot more about the powerful features of JavaScript through this project such as the ability to dynamically generate html tags through methods such as ".appendChild"!
