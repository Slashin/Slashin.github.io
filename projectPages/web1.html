This was the project within which I was able to go from being just a front-end web developer to being a full-stack web applications developer. The entire application is built with the MERN stack, while using Firebase for authentication and the IBM Watson API for the speech-to-text transcription. The application is currently in alpha stage and has received positive feedback from over 20 early testers! I am currently in the process of gathering more users and trying to recieve funding!

Tools Used: HTML, CSS, JavaScript, React.js, Node.js, MongoDB, Express.js, FireBase, IBM Watson API
