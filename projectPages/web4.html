Select'N Search

Previous Next

This project was a very hasty one. My teammates and I created it within only 3 hours and submitted it to the Waterloo Enghacks Hackathon just a couple minutes before the deadline, but managed to still secure first place! "Select'N Search" is a very useful chrome extension that allows the user to simply select any amount of text required and with the press of a button query that text on multiple websites. Based on the specfic key that is pressed, the program will lead the user to their destination site and search the text on that site. It is a very efficient and time-saving application that can truly benefit many people. Check out the Chrome Web Store link below!

Tools Used: HTML, CSS, JavaScript, JSON
